# Screenshots & preview image

Add visuals here to make the repo and Workshop listing shine.

## For the README

Drop a couple of PNGs in this folder and reference them, e.g.:

```markdown
![The AI's in-game HUD](docs/screenshots/hud.png)
![The live mind dashboard](docs/screenshots/mind.png)
```

Good ones to capture:
- **`hud.png`** — the in-game HUD (bottom-left) mid-decision, ideally with a speech bubble and a zombie in frame.
- **`mind.png`** — the web dashboard at `http://localhost:8799/mind` showing a thought + vitals.
- **`fight.png`** — the survivor mid-swing (`FIGHT` on the HUD) for the action shot.
- An animated **`demo.gif`** of a short run is the single best thing you can add.

## For the Steam Workshop preview

Steam wants a **`preview.png`** in the `workshop/` folder, ~512×512. A clean shot of the HUD over
a gameplay moment works well. Keep text legible at thumbnail size.

## How to grab a clean shot

Run in borderless-windowed (Options → Display) so normal screenshot tools work, or use PZ's own
screenshot key. Crop to 16:9 for the README, square for the Workshop preview.
